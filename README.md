# WebChat - BackEnd

**Para executar o projeto:**

**1 - Com docker-composer**

```bash
docker-compose up
```

**OU**


**2 - Sistemas Linux**

- Configure as variáveis de ambiente

```bash
.env
```

- Instale as dependências:
```bash
npm install
```

- Inicialize o projeto:
```bash
npm start
```


Deploy ~> https://webchat-vool-backend.herokuapp.com/
