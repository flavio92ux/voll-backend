# WebChat - BackEnd

Para executar o projeto:

**docker (recomendado)**
- Rode o servidor:
```bash
docker-compose up
```

**localmente**

- Configure as variáveis de ambiente;

- Instale as dependências:
```bash
npm install
```

- Rode o servidor:
```bash
npm start
```


Deploy ~> https://webchat-vool-backend.herokuapp.com/
