# WebChat - BackEnd

Para executar o projeto:

**docker (recomendado)**

```bash
docker-compose up
```

**Rodando localmente**

- Configure as variáveis de ambiente;

```bash
.env
```

- Instale as dependências:
```bash
npm install
```

- Inicialize o projeto:
```bash
npm start
```


Deploy ~> https://webchat-vool-backend.herokuapp.com/
